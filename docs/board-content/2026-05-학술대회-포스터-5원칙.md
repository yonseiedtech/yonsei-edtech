# 처음 학술대회 포스터를 만든다면 — 5가지 핵심 원칙

> 카테고리: 학술 활동 가이드 / board/free 또는 신설 카테고리
> 작성: 운영진 / 2026-05-12

학술대회 포스터 발표는 학위 과정 중 가장 가성비 좋은 학습 경험 중 하나입니다. 청자와 1:1 토론 기회가 많고, 본인 연구의 약점을 가장 빠르게 발견하게 해주기 때문입니다. 다만 처음 포스터를 만들 때 흔히 빠지는 함정이 있어 — 5가지 원칙으로 정리합니다.

## 원칙 1. **3미터 거리에서 제목이 읽혀야 한다**
학회장 포스터 세션은 사람이 분주하게 오가는 환경입니다. **3미터 떨어진 곳에서 제목과 핵심 결론이 안 보이면 청자가 멈추지 않습니다.**
- 권장: 제목 글꼴 ≥ 90pt, 본문 ≥ 24pt
- 실수: PowerPoint 기본 폰트(18pt)를 그대로 사용 → 청자 0명

## 원칙 2. **포스터 ≠ 논문 축약본**
A0 사이즈에 논문 모든 문장을 빽빽이 넣는 것은 가장 흔한 실수입니다. 포스터는 **"30초 안에 본인 연구가 무엇인지 전달하기 위한 시각 자료"**이지 논문 압축본이 아닙니다.
- 권장 구성: 제목 / 연구 질문 1줄 / 그림 1개 / 핵심 결론 2~3줄
- 텍스트는 청자가 본인과 대화하는 동안 보조 자료로만 쓰이도록.

## 원칙 3. **시각 요소를 본문 위에**
선행 연구가 강조하는 정보 위계는 명확합니다: **이미지 > 차트 > 짧은 문장 > 긴 문장**. 청자의 시선은 그래픽으로 먼저 향합니다.
- 본인 연구 결과의 핵심 그래프 1개를 포스터의 시각적 중심에 배치
- 본인이 직접 만든 다이어그램이 있다면 더 좋음 (출처가 자기 자신인 시각 자료가 가장 강력)

## 원칙 4. **이야기 흐름이 좌→우 / 위→아래 자연스럽게**
다국가 학회에서도 통하는 표준은 **좌상단(연구질문) → 우상단(방법) → 좌하단(결과) → 우하단(시사점·앞으로의 연구)**.
- 청자가 본인 포스터 앞에서 어디부터 봐야 할지 헷갈리면 다른 포스터로 갑니다.
- 화살표·번호로 흐름을 명시해주면 더 친절.

## 원칙 5. **20초·1분·5분 세 가지 발표문을 준비한다**
포스터 세션 중에는 청자에 따라 머무는 시간이 천차만별입니다. 미리 세 가지 길이의 본인 발표문을 연습해두세요.
- **20초**: 제목 + 연구 질문 + 핵심 결론 1줄 (지나가는 청자용)
- **1분**: 위 + 방법 1줄 + 시사점 1줄 (관심 보이는 청자용)
- **5분**: 위 + 그래프 해석 + 한계점 (질문 많은 청자·심사위원용)

## 부록: 자료·도구
- 포스터 디자인 도구: PowerPoint, Keynote, Canva(템플릿), LaTeX(beamerposter)
- 인쇄: 학회장 근처 인쇄소, 또는 학회 자체 인쇄 서비스(많은 학회가 사전 신청 시 무료 제공)
- 본인 포스터를 발표 후 [에듀테크 아카이브](/edutech-archive)에 등록하시면 동료들의 참고 자료가 됩니다.
