# 교육공학 이론이 학회 서비스에 스며들다 — 7가지 적용 사례

> 카테고리: 자료실 또는 자유게시판 / 운영진 작성
> 작성: 운영진 / 2026-05-13

연세교육공학회는 학회 사이트 자체가 교육공학 이론이 작동하는 살아있는 학습 환경이 되도록 설계했습니다. 이번 업데이트로 7가지 교육공학 이론이 회원 경험에 직접 반영되었습니다. 어떤 이론이 어떤 기능에 어떻게 스며들었는지 정리합니다.

## 1. Bloom's Taxonomy (Anderson & Krathwohl, 2001) — 디딤판 학기별 카드

학기별 로드맵 카드마다 본 학기에 요구되는 주된 인지 활동 단계를 명시합니다. 1학기차는 "이해", 4학기차는 "평가", 디펜스 학기는 "창조" 등 — 학습자가 본인이 현재 어떤 인지 활동에 집중해야 하는지 메타인지적으로 자각할 수 있습니다.

## 2. Kolb's Experiential Learning Cycle (Kolb, 1984) — 분석 노트 4단계 템플릿

학술대회 후기·세미나 회고 작성 시 "Kolb 4단계 템플릿 사용" 토글을 켤 수 있습니다. 구체적 경험 → 반성적 관찰 → 추상적 개념화 → 능동적 실험의 4단계로 본인 경험을 구조화할 수 있도록 가이드 패널이 펼쳐집니다.

## 3. Keller's ARCS Motivation Model (Keller, 1987) — 마이페이지 동기 패널

본인 학회 참여를 Attention·Relevance·Confidence·Satisfaction 4축으로 점수화하여 시각화합니다. 약한 축에는 자동으로 보강 안내가 표시되어 — 예를 들어 관심 키워드가 적으면 "프로필에 관심 분야를 추가하세요" 액션이 제안됩니다.

## 4. Cognitive Apprenticeship (Collins, Brown & Newman, 1989) — 인터뷰 게시판

인터뷰 게시판은 선배·졸업생의 암묵지를 명시적으로 노출하는 디지털 인지 도제 환경입니다. 페이지 상단에 이 이론적 맥락을 명시하여 회원이 단순히 글을 읽는 것이 아니라 modeling → scaffolding → fading의 학습 도제 관계로 접근하도록 유도합니다.

## 5. Spaced Repetition (Ebbinghaus 1885 + Cepeda et al. 2008) — 대시보드 위젯

본인이 작성한 게시글을 1·7·14·30일 간격으로 다시 안내합니다. Cepeda et al.(2008) 메타분석에 따르면 이 간격이 장기 기억 형성에 가장 효과적입니다. 본인이 쓴 글을 시간 간격을 두고 다시 보면 본인의 사고가 어떻게 발전했는지 자가 점검할 수 있습니다.

## 6. Microlearning + Reflective Practice (Hug 2005 / Schön 1983) — 매일 5분 회고

대시보드 상단에 매일 다른 회고 프롬프트가 노출됩니다. 메타인지·전이·비판적 사고·Bloom 분석·Kolb 실험 등 14개 프롬프트가 본인 uid와 날짜를 시드로 매일 선택됩니다. 5분 회고를 누적하면 메타인지가 강화됩니다.

## 7. Connectivism (Siemens, 2005) — 네트워크 학습 시각화

학회 네트워킹 맵에서 동기·신분 유형·학교급 차원으로 본인의 학습 네트워크를 시각화합니다. Siemens 이론에 따르면 학습은 노드 간 연결을 형성·유지하는 능력 그 자체입니다. 본인이 누구와 학습 자원을 공유할 수 있는지 한눈에 파악하고 네트워크 확장 기회를 발견할 수 있습니다.

## 왜 이론을 사용자 UX 에 직접 반영하는가

전통적으로 교육공학 이론은 학자가 연구하고 교사가 적용하는 형태입니다. 그러나 본 학회 사이트는 회원이 본인의 학습 경험을 메타인지적으로 자각할 수 있도록 이론을 UX 라벨·구조·인터랙션으로 직접 노출합니다.

이러한 접근은 두 가지 효과를 노립니다.

첫째는 학회의 학문적 정체성 강화입니다. 단순한 학회 사이트가 아니라 교육공학 이론이 작동하는 환경 자체가 우리의 차별화 자산이 됩니다.

둘째는 회원의 메타인지 자가 학습 강화입니다. 본인이 어떤 인지 활동을 하고 있는지(Bloom), 어떤 경험학습 단계에 있는지(Kolb), 어떤 동기 차원에 강점·공백이 있는지(ARCS), 학습 네트워크의 어디에 위치하는지(Connectivism) 등 — 평소에는 보이지 않는 학습의 구조를 시각화합니다.

## 후속 계획

- Mastery Learning (Bloom, 1968) 진행률 추적 — 디딤판 항목별 완료 체크리스트

- Self-Determination Theory — 자율성·유능감·관계성 통합 패널
