# 디딤판 체크리스트로 본인 학기 마스터하기 — Mastery Learning 적용 가이드

> 카테고리: 자유게시판 또는 자료실 / 운영진 작성
> 작성: 운영진 / 2026-05-13

연세교육공학회 [인지디딤판](/steppingstone) 에 최근 학기별 항목 체크리스트와 진행률 표시 기능이 추가되었습니다. 단순히 항목을 읽는 것에서 넘어 본인이 직접 체크하면서 본인 학기를 마스터하는 도구로 진화한 것입니다. 본 기능의 이론적 배경과 활용법을 정리합니다.

## 이론 배경 — Mastery Learning (Bloom, 1968)

Mastery Learning 은 Benjamin Bloom 이 1968년 제안한 교수설계 원리입니다. 핵심 명제는 다음과 같습니다.

거의 모든 학습자는 충분한 시간과 적절한 안내가 주어지면 어떤 학습 단위든 숙달할 수 있다.

전통적 학습이 시간을 고정하고 결과(성취도)를 변동값으로 두는 것과 반대로, Mastery Learning 은 결과(숙달)를 고정하고 시간을 변동값으로 둡니다. 한 단위를 완전히 숙달한 후에야 다음 단위로 이동합니다.

본 학회 디딤판의 항목 체크리스트는 이 원리를 디지털화한 것입니다. 학기별 5개 항목을 본인이 직접 체크하면서 본인의 숙달 진행률을 시각화합니다.

## 활용법 5단계

### 1) 학기 시작 첫 주에 본인 학기 카드 점검

학기 시작 첫 주에 [디딤판](/steppingstone) 에 들어가시면 본인 학기에 해당하는 카드가 자동으로 강조됩니다. 5개 항목을 천천히 읽으시고 본인이 이미 완료한 항목이 있다면 체크박스를 눌러 표시하세요.

### 2) 미완료 항목을 본인 학기 목표로 설정

체크하지 않은 항목을 본인 학기의 목표로 삼으세요. 예를 들어 1학기차 학생이 "지도교수님 정하기" 가 미완료라면 학기 중반까지 완료를 목표로 분석 노트에 기록해두세요.

### 3) 항목별 완료 시 즉시 체크

목표 항목을 완료한 즉시 체크하세요. 체크와 동시에 카드 상단 진행률 (예: "3/5 완료") 와 progress bar 가 업데이트됩니다. 본인 진행률이 시각적으로 변하는 것은 작은 도파민 보상으로 작용해 다음 항목 진행 동기로 이어집니다.

### 4) 100% 완료 시 Mastery Achieved 배지 확인

5개 항목을 모두 완료하면 카드 우상단에 Trophy 아이콘과 함께 "Mastery Achieved" 배지가 나타납니다. 본인이 본 학기를 마스터했다는 시각적 확인입니다. 본인 성취감을 그대로 분석 노트에 기록해두세요. 다음 학기에 같은 패턴으로 진행하실 때 본인 효능감의 근거가 됩니다.

### 5) 다음 학기 카드 미리 검토

본인 학기를 마스터한 직후, 다음 학기 카드도 미리 검토해보세요. 미리 시작할 수 있는 항목 (예: 4학기차에 5학기차 디펜스 연습 도구 등록하기) 이 있다면 본인 학기 마스터 후 자연스러운 다음 단계로 이어집니다.

## 주의사항

본 체크리스트는 본인의 브라우저 로컬 저장소(localStorage) 에 보관됩니다. 같은 계정이라도 다른 기기 (집 노트북·휴대폰·연구실 PC) 에서는 진행률이 동기화되지 않습니다. 학회 운영진은 다음 사이클에 Firestore 동기화를 검토하고 있으나, 현재는 본인이 주로 사용하는 한 기기에서 추적하시는 것을 권장합니다.

또한 체크리스트는 본인 자가 평가 도구입니다. "지도교수님 정하기" 같은 항목이 본인 상황에 맞지 않으면 굳이 체크하지 않으셔도 됩니다. 디딤판 운영진은 항목을 일반론으로 제공할 뿐, 본인 경로의 정답을 강요하지 않습니다.
