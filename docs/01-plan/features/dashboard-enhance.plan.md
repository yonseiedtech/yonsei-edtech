# Plan: 대시보드 고도화 (dashboard-enhance)

## 1. 개요

방금 생성한 `/dashboard` 페이지를 고도화하여
실제 데이터 연동, 개인화 콘텐츠, 빠른 액션을 강화한다.
bkend 연동 이후 단계에서 진행.

## 2. 목표

- Mock 데이터 → 실제 API 데이터 표시
- 역할별 맞춤 대시보드 콘텐츠
- 최근 활동 피드 (내 댓글, 좋아요 등)
- 알림 배지 (새 공지, 세미나 변경 등)
- 반응형 개선 (모바일 최적화)

## 3. 사용자 스토리

| 역할 | 스토리 | 우선순위 |
|------|--------|----------|
| 회원 | 로그인하면 내 활동 요약을 한눈에 볼 수 있다 | P0 |
| 회원 | 읽지 않은 공지사항 수를 확인할 수 있다 | P1 |
| 회원 | 다가오는 세미나 일정을 달력 형태로 볼 수 있다 | P2 |
| 운영진 | 승인 대기 회원 수 + 미답변 문의 수를 실시간으로 확인한다 | P1 |
| 운영진 | 대시보드에서 바로 승인/답변 페이지로 이동할 수 있다 | P1 |

## 4. 기능 범위

### In Scope
- **StatCard 실 데이터 연동**: bkend API에서 카운트 조회
- **역할별 위젯**: staff+는 관리 위젯, member는 학습 위젯
- **최근 활동 피드**: 내가 작성한 글의 새 댓글 알림
- **캘린더 미니 위젯**: 이번 달 세미나 일정 표시
- **모바일 레이아웃**: 1열 스택 + 스와이프 카드

### Out of Scope (별도 피처)
- 실시간 푸시 알림 (알림 시스템)
- 대시보드 커스터마이징 (위젯 추가/제거)
- 다크모드

## 5. 기술 설계 개요

### 5.1 위젯 구조

```
DashboardContent
  ├── WelcomeSection (이름 + 역할 + 기수)
  ├── StatCards (4열 → API 데이터)
  ├── QuickActions (역할별 동적)
  ├── Grid 2열
  │   ├── RecentNotices (5건)
  │   └── UpcomingSeminars (2건) 또는 MiniCalendar
  ├── ActivityFeed (내 글의 새 댓글 등)
  └── MySeminars (신청 세미나 요약)
```

### 5.2 의존 피처

- `bkend-integration` 완료 후 진행 가능
- 캘린더 위젯은 별도 라이브러리 필요 없음 (자체 구현)

## 6. 예상 작업량

- **M (Medium)** — 2세션
- bkend 연동 후 StatCard 데이터: 0.5세션
- 역할별 위젯 + 활동 피드: 1세션
- 캘린더 미니 위젯: 0.5세션

## 7. 선행 조건

- `bkend-integration` 피처 완료 필수
- 실 데이터 없이는 의미 있는 고도화 불가
