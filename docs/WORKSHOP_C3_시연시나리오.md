## 시연 흐름 (28분)

### 0. 오프닝 — "논문은 여정이다" (2분)
- 대시보드 → **논문 여정 7단계 지도** 보여주며: "오늘 보여드릴 도구는 이 여정의 앞 4단계를 그대로 따라갑니다 — 주제를 잡고, 문헌을 쌓고, 모형을 그리고, 계획서까지."
- 멘트 포인트: "각 단계 산출물이 다음 단계에 자동으로 이어집니다. 따로 노는 도구 5개가 아니라 하나의 파이프라인이에요."

### 1. 주제 탐색 인터뷰 (7분) — `/mypage/research?tab=explore`
1. 질문 7개를 실제로 답하며 진행 — **현장 세부(초/중/고/특수, 군/공기업 등)와 관심 세부(생성형 AI, 자기조절 등)를 고르는 장면**을 천천히: "질문이 구체적일수록 추천 문장도 구체적으로 나옵니다."
2. 결과 화면에서 순서대로:
   - 추천 주제 문장 → **복사 버튼** 클릭: "이 문장은 완성이 아니라 출발점 — 내 현장 단어로 바꾸세요."
   - **개념 이해도 확인**: 약점 배지 + "진단평가로 확인하기" — "주제를 잡자마자 내가 뭘 모르는지부터 알 수 있어요."
   - 비슷한 조건의 **선배 논문**: "제목만 훑어도 주제가 문장이 되는 방식이 보입니다."
   - 상단 **비교하기**: 다른 답으로 한 번 더 탐색한 뒤 두 결과 나란히 비교 시연.
- 멘트 포인트: "탐색할 때마다 일시와 함께 자동 저장되니, 지도교수님 면담 전에 여러 방향을 만들어 비교해 가세요."

### 2. 문헌 매트릭스 + DOI 자동 채움 (8분) — `/mypage/research?tab=reading&focus=matrix`
1. 논문 등록 → **DOI만 입력하고 '자동 채움'**: 제목·저자·연도·저널이 채워지는 순간이 이 세션의 하이라이트. "서지 입력에 쓰는 시간을 0으로."
2. 분석 노트 카드에서 **APA 인용 복사** 원클릭 시연.
3. 매트릭스로 스크롤 — 방금 등록한 논문의 **대상·방법·주요 결과 셀**을 즉석에서 1~2칸 채우기: "논문 5편만 이렇게 쌓이면 선행연구 표가 저절로 나옵니다."
4. 잔디(+2) 반영 토스트 보여주기: "정리할 때마다 학습 잔디에 쌓입니다."
- 트러블슈팅: DOI 조회 실패 시 → 직접 입력 탭으로 자연스럽게 전환 ("해외 DB 지연일 수 있어요").

### 3. 연구 모형 마법사 (5분) — `/research-model`
1. 마법사로 독립변인 → 종속변인 연결, 매개/조절 1개 추가.
2. 완성 모형 저장: "매트릭스에서 반복해 나온 변인들이 그대로 모형 재료입니다."
- 멘트 포인트(전공 관례): "모형은 분석 설계를 돕는 도구입니다 — 본문에 가설을 직접 쓰는 형식이 아니라는 점은 지도교수님 지도에 따르세요."

### 4. 계획서 인터뷰 → 논문 에디터 시딩 (6분) — `/mypage/research?tab=proposal`
1. 연구계획서 **인터뷰 모드** 진입 — 챕터별 질문에 답하면 문서가 조립되는 구조 1~2문항만 시연.
2. 논문 작성 탭(`tab=writing`) → 에디터의 **'계획서에서 가져오기'**: "계획서가 통과되면 그 내용이 논문 1~3장의 뼈대로 — 빈 장에만 안전하게 들어갑니다."
3. 마무리로 연구 코크핏(장별 진행 게이지) 한 번 보여주기: "쓰다 보면 내가 어디까지 왔는지 이 계기판이 알려줍니다."

## 실습 미션 안내 (3분)

> "오늘 집에 가서 딱 세 가지만 해보세요 — 대시보드 새 회원 체크리스트에 그대로 떠 있습니다."

1. **주제 탐색 1회** 돌리고 마음에 드는 문장 저장 (자동 저장됨)
2. **문헌 매트릭스에 논문 1편** 정리 (DOI 자동 채움 사용)
3. **연구 모형 1개** 저장

- 완료하면 온보딩 체크리스트가 채워지고 잔디·리더보드에 반영됨을 안내.
- 다음 세미나에서 "매트릭스 5편 채운 분" 소개 예고 (소셜 프루프).

## 예상 Q&A (4분)

| 질문 | 답 |
|---|---|
| 질적/혼합연구인데 쓸 수 있나요? | 주제 탐색·계획서 인터뷰 모두 접근별 분기 지원 — 질적이면 질적 프레임으로 추천·구성됩니다. |
| 모바일로도 되나요? | 전부 반응형. 매트릭스처럼 넓은 표는 가로 스크롤. |
| 제 데이터 누가 보나요? | 연구활동은 본인 전용, 운영진은 지원 목적의 열람만 가능(수정 불가). |
| 진단평가는 몇 문항? | 개념 단위 진단 — 주제 탐색 결과에서 약점 개념부터 확인 권장. |
