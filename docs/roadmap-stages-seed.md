# 학기별 로드맵 — Firestore 초기 시드 가이드

> 운영진이 `/console/roadmap`에서 한 번에 6단계를 등록할 때 그대로 복사해서 사용하세요.
> 작성: 2026-05-12 (Sprint 67-AR)

---

## 등록 절차

1. `https://yonsei-edtech.vercel.app/console/roadmap` 접속 (staff 이상 권한 필요)
2. "새 단계 추가" 클릭
3. 아래 6개 블록을 하나씩 그대로 입력 → "등록" 반복
4. 각 단계 등록 후 **공개** 체크박스를 켜야 회원에게 노출됨

---

## 1단계 — 1학기차 (정착)

- **순서**: 1
- **매칭 학기**: 1
- **제목**: `1학기차 — 적응과 시작`
- **짧은 태그**: `정착`
- **색상**: 파랑
- **졸업 후 단계**: ❌
- **공개**: ✅
- **체크리스트 항목**:
```
신입생 OT 참여 + 학회 가입 신청
지도교수님 정하기 (1학기 말 권장)
교육공학 핵심 과목 (교수설계론·학습이론) 수강
세미나 정기 참여 — 학회 분위기 익히기
동기 명함 교환 + 네트워크 형성
```

## 향후 보강 권장

본 시드는 현재 디딤판 정적 콘텐츠와 동일합니다. 운영진이 이후 다음을 추가하시면 학회 특수성이 강화됩니다:

- **연세교육공학 특화**: 김교수님 연구실 / 박교수님 수업명 / 학사 마감일 등
- **링크 추가** (현재 UI 미지원, 향후 확장 예정): 항목별 `[지도교수 디렉토리](/directory)` 형식
- **학기별 학사일정 자동 연동**: 수강신청 기간·논문 제출 마감 등

운영진이 이 자료를 매년 학기 시작 시 점검하면 디딤판이 살아있는 콘텐츠로 유지됩니다.
