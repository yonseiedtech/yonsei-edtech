# 연구 여정 v2 제안서 — 보고서→계획서→논문 커버리지 분석과 발전 방향

작성일: 2026-07-03 · 근거: 세 에디터(ResearchReport/Proposal/WritingPaper) 전 필드 코드 분석

## 0. 요약 판정

세 단계의 **뼈대와 연결 허브(연구모형·연구문제·변인 동기화)는 이미 우수**하다. 그러나 ① 보고서의 풍부한 산출물(이론 카드·선행연구 분석·환경/학습자 분석)이 계획서·논문으로 **자동 이관되지 않고**, ② 석사논문 필수 요건 중 **3건(P0)이 어느 단계에도 없으며**, ③ 여정 지도가 2벌(3단계 가이드 vs 5단계 ThesisJourney)로 병존한다.

## 1. 커버리지 현황 (핵심만)

**보고서 → 계획서**: 범위(scope)·연구문제·변인은 연결됨(모형 허브 경유) / 제목·목적·방법은 부분 / **6챕터 본문(현장 문제·환경·학습자·이론·선행연구)은 이관 코드 없음** — 사용자가 수기 재작성.

**계획서 → 논문**: seedFromProposal이 목적·범위·방법 **3개 필드만** 시딩. 연구문제 태그→통계 골격 연결(③→④)은 우수. **보고서의 이론 카드→이론적 배경, 선행연구 분석→선행연구 절 이관 없음.**

## 2. 갭 (석사논문 표준 요건 대비)

**P0 — 필수인데 완전 부재**
1. **연구윤리/IRB/동의서**: 필드·가이드·체크리스트 전무 (부록 예시 문구뿐)
2. **용어의 정의(조작적 정의)**: 서론 절 자체가 템플릿에 없음. 변인별 조작적 정의 구조 없음
3. **연구 가설**: RQ만 있고 가설(방향 포함) 구조 없음 — "연구모형 및 가설" 절은 빈 템플릿

**P1 — 심사 단골 요구**
4. 선행연구 **비교표**(저자·연도·대상·설계·결과 매트릭스) — 문단·그룹 구조만 존재
5. 측정도구 **신뢰도·타당도 표**(도구별 α) — 가이드 팁뿐
6. **연구 절차 타임라인**(사전-처치-사후, 주차) — 방법장에 절 없음
7. **논의(Discussion) 독립 절** — 결론에 흡수되어 선행연구 비교 논의 구조 없음

**P2 — 형식**: 영문 초록, 참고문헌 절(본문), 목차/표목차, 감사의 글

## 3. 발전 프로젝트 제안 (우선순위)

### R1. 논문 템플릿 보강 (공수 小 — 즉시)
- 서론에 "용어의 정의" 절, 방법에 "연구 절차", 결론을 "논의 → 결론 → 제언(한계·후속연구)"로 재편(또는 논의 절 추가)
- AbstractPanel에 영문 초록 토글 추가

### R2. 가설 구조화 + 윤리 체크리스트 (공수 中)
- ResearchQuestionItem에 `hypothesis?: {text, direction}` 확장 — 연구문제↔가설 쌍 관리, "연구모형 및 가설" 절에 자동 나열
- 방법 장에 IRB/동의/개인정보 체크리스트 패널(보고 문형 포함) — 지도노트·부록과 연계

### R3. 상류→하류 이관 파이프 (공수 中 — 가장 체감 큰 항목)
- "보고서에서 가져오기" 버튼: 이론 카드→이론적 배경 절 초안, priorResearchAnalysis→선행연구 절, 현장 문제(현상·심각성)→연구의 필요성 초안
- 계획서 referencePaperIds → 논문 "참고문헌" 절 자동 생성(APA7 재사용)

### R4. 문헌 리뷰 매트릭스 (공수 中大 — 새 도구)
- 읽은 논문(ResearchPaper)을 행으로, 저자·연도·대상·설계·주요결과·시사점을 열로 하는 비교표 편집기
- 보고서 6장(선행연구)과 논문 이론적 배경에 표로 삽입 가능 → P1-4 해소 + 논문 읽기 기능의 가치 승격

### R5. 도구 신뢰도 표 + 절차 타임라인 위젯 (공수 中)
- 측정도구 절: 도구명·문항수·출처·α(선행/본연구) 행 편집기 → 표 삽입
- 연구 절차: 주차·활동 타임라인 입력 → 절차 절 표/문장 생성

### R6. 여정 지도 통합 + 단계 확장 (공수 中)
- ResearchJourneyGuide(3단계)를 ThesisJourney(5학기) 위에 정렬된 단일 지도로 통합
- 중간 단계 3개 삽입: **문헌 매트릭스**(주제탐색↔보고서 사이, R4와 연동) → **윤리·도구 확정**(계획서↔논문 사이, R2·R5와 연동) → **심사 대응·수정**(논문 뒤 — 지도노트를 데이터 백본으로, 심사 연습 도구 연결)
- 최종 여정: 주제 탐색 → 문헌 매트릭스 → **연구 보고서** → **연구 계획서** → 윤리·도구 확정 → **논문 집필** → 심사 대응·수정

### 권장 순서
R1(템플릿, 반나절) → R3(이관 파이프, 체감 최대) → R2(가설·윤리) → R6(여정 통합) → R4·R5(새 도구).
