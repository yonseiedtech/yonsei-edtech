# 📎 논문 도구 워크숍 핸드아웃 — 오늘 배운 것, 경로 한 장 정리

> 로그인: https://yonsei-edtech.vercel.app (회원 승인 후 이용 가능)

## 오늘의 파이프라인

| 단계 | 도구 | 경로 | 이런 걸 해줍니다 |
|---|---|---|---|
| ① 주제 잡기 | **주제 탐색 인터뷰** | 내 연구활동 → 주제 탐색 | 현장·관심·상황 7문항 → 주제 문장 추천, 자동 저장·비교 |
| ② 개념 점검 | **진단평가** | 주제 탐색 결과 → "이해도 확인하기" | 추천 주제 주변 개념 중 내 약점 표시 |
| ③ 문헌 쌓기 | **문헌 매트릭스** | 내 연구활동 → 논문 읽기(문헌고찰) | DOI 자동 채움, APA 복사, 선행연구 비교표 |
| ④ 모형 그리기 | **연구 모형 마법사** | /research-model | 변인 연결로 연구 모형 도식화 |
| ⑤ 계획서 | **계획서 인터뷰 모드** | 내 연구활동 → 연구계획서 | 질문에 답하면 계획서가 조립됨 |
| ⑥ 논문 | **논문 에디터** | 내 연구활동 → 논문 작성 | '계획서에서 가져오기'로 1~3장 뼈대 시딩 |

## 🎯 이번 주 미션 (대시보드 체크리스트에 떠 있어요)

- [ ] 주제 탐색 1회 — 마음에 드는 문장이 나올 때까지 비교해보기
- [ ] 문헌 매트릭스에 논문 1편 정리 — DOI 자동 채움 써보기 (잔디 +2)
- [ ] 연구 모형 1개 저장

## 💡 소소한 팁

- 주제 탐색 결과의 **"복사"** 버튼 → 지도교수님 면담 메모에 붙여넣고 내 말로 다듬기
- 논문 카드에 **읽기 타이머**를 걸면 읽은 시간이 자동 기록 (30분+ 잔디 +3)
- 매트릭스 **'APA 내보내기'** — 지금까지 정리한 논문이 가나다순 참고문헌으로
- 막히면: 도움말(/help) → 그래도 안 되면 문의(/contact) 또는 소통 보드
