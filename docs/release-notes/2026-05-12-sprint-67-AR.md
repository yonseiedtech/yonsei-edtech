## 🧭 인지디딤판 콘텐츠 강화

지난 외부 피드백에서 가장 많이 지적되었던 부분이 **"디딤판은 노출은 강조되었으나 안에 들어갈 콘텐츠가 부족하다"** 였습니다. 이번 업데이트로 디딤판이 **4개 트랙 모두 즉시 진입 가능**한 상태가 되었습니다.

### 신설된 페이지
- **재학생 학습 가이드** (`/steppingstone/current-student`) — 학기별 학습 흐름, 연구 노하우, 세미나 활용법, 필수 체크리스트
- **학술대회 대비** (`/steppingstone/conference`) — 국내·국제 주요 학회 일정, 5단계 발표 신청 가이드, 발표 후 자산화

### 학기별 로드맵 자동 안내
- 디딤판 메인에 **본인 학기에 해당하는 카드가 자동으로 강조**되는 학기별 로드맵을 추가했습니다.
- 1학기차(정착) → 2학기차(탐색) → 3학기차(본격) → 4학기차(집필) → 디펜스 학기(심사) → 졸업 후(동문) — 6단계
- 로그인하면 "내 학기" 배지가 본인 카드 위에 표시됩니다.

## 👋 신규 가입 환영 흐름

신규 회원이 가입 후 곧바로 가치 있는 첫 화면을 경험할 수 있도록 두 가지를 보강했습니다.

1. **자동승인 가입 시 환영 화면 분기** — `@yonsei.ac.kr` 등 사전 승인 도메인은 가입 즉시 환영 + 디딤판 진입 안내가 나타납니다.
2. **대시보드 환영 배너** — 가입 후 7일 이내 회원에게 대시보드 상단에 "본인 학기 가이드가 준비되어 있어요" 배너가 표시됩니다. 닫기 가능.

## 🧹 마이페이지 진입점 통합

기존 헤더 메뉴에 분산되어 있던 "개인 페이지 · 내 학회활동 · 내 연구활동 · 내 명함" 4개 항목을 통합하고, **"마이페이지" 1개 진입점**으로 정리했습니다. 모바일·데스크톱 동일하게 적용되었습니다. 명함·활동·연구는 마이페이지 내부 탭으로 흡수 예정(차기 업데이트).
