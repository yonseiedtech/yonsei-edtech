# 역할 권한 매트릭스

## 역할 계층

| 역할 | 코드 | 설명 |
|------|------|------|
| 관리자 | `admin` | 시스템 전체 관리 |
| 회장 | `president` | 학회 운영 총괄 |
| 운영진 | `staff` | 게시글·세미나 관리 |
| 자문위원 | `advisor` | 자문위원 (졸업생과 동일 레벨) |
| 졸업생 | `alumni` | 졸업 회원 |
| 회원 | `member` | 일반 학회원 |
| 게스트 | `guest` | 비로그인 (user === null) |
