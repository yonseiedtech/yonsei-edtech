## 1차 작업: 그리드 외 chip 노출 (b924e768)

### 문제 (사용자 보고)
> "이번주 수업에 타임라인에 '스터디' 일정이 반영이 제대로 여전히 안돼"

### 원인 (`src/features/dashboard/DailyClassTimelineWidget.tsx`)
학술활동(스터디·프로젝트·대외) 진행 일정이 3가지 경로로 무음 누락:
1. **토·일 활동**: `WEEK_DAY_INDICES = [1,2,3,4,5]` (월~금만) → 주말 일정 continue
2. **시간 17~23 범위 밖**: `e <= s` → continue
3. **시간 미정**: 정상 chip 표시 (기존 동작)

→ 17시 이전·토요일 스터디는 그리드에도 chip에도 안 보임.

### 해결
`undatedWeeklyActivities`/`undatedDailyActivities` 로직 확장:
- 주말·시간미정·시간외 3가지 모두 chip으로 노출
- 각 chip에 reason 배지 부착 (주말/시간 미정/시간표 시간외)
- WeeklyGrid 안내문: "주말(토·일) · 시간 미정 · 시간표 시간외 항목 — 시간표 시간 범위를 넓히면 그리드에 표시됩니다."
