## 0. 요약 (TL;DR)

- 참고 자료 2종은 90%가 **"인용하는 행위" 자체**(직접/간접 인용, 표절 회피, 내 말로 쓰기, 저자-연도 표기, 재인용 지양)에 관한 내용이다. 선행연구 정리·서론·방법론은 자료의 후반부지만 **연계 작업 소관이므로 본 가이드에서 제외**한다.
- 기존 아카이브에는 `/archive/apa-style`(정적, **참고문헌 형식** 위주)가 이미 있다. 그러나 **본문 인용을 "내 말로 바꾸는 방법"·표절 회피·직접/간접 인용 구분·재인용**을 정면으로 다루는 가이드는 없다 → 신규 독립 가이드가 정당하다.
- 권장 구현: `paper-guide`·`apa-style`와 동일한 **정적(static) 단일 페이지** 패턴(`PageContainer` + `PageHeader` + `ArchiveStickyToc` + 섹션 데이터 배열). archive-seed/Firestore 신규 컬렉션 불필요. 기존 `apa-style`는 **참고문헌 형식 레퍼런스**로 역할을 좁히고, 신규 가이드에서 상호 링크로 연계.

## 1. 자료에서 추출한 인용 핵심·예시 (출처 표시)

> 출처 약칭: **[리포트]** = `260610_채찍질_석사방 - 자세한 리포트.md`, **[스크립트]** = `스크립트.txt`. 두 자료는 동일 강의의 정리본/녹취록이며 내용이 일치한다. 자료 내 모든 인용 형식 예시는 **가상 저자(홍길동·이연구 등)** 기반이다.

### 1-A. 인용의 본질·필요성 (가이드 도입부 근거)
- 논문은 "선배 연구자들의 어깨 위에 올라서서" 한 발 더 나아가는 선언이며, 인용은 그들의 작업을 공식 인정하고 **내 주장의 근거**를 만드는 행위. [리포트 §1.1 / 스크립트]
- 선행연구 없는 논문은 존재할 수 없고, 근거 없는 주장은 "블로그 주장"에 불과. [스크립트]
- 잘못된 인용 = 표절 = 학문적 윤리 위반. (강의에서 강조: 본인/가족의 미래 검증 리스크) [리포트 §1.1 / 스크립트]

### 1-B. 표절의 정의·유형 (반드시 피해야 할 패턴)
- **유형 ① 출처 미표기**: 타인 연구 결과를 자기 발견처럼 출처 없이 서술. [리포트 §1.2]
- **유형 ② 원문 복붙/유의어 치환**: 원문을 그대로 또는 거의 그대로 가져오기. **출처를 달았더라도 문장 구조가 동일하고 단어만 유의어로 바꾸면 표절로 간주** — 이 기준이 매우 엄격함. [리포트 §1.2 / 스크립트]
- 핵심 원칙: "단어만 바꾸는 게 아니라 **문장 구조를 완전히 바꾸고 내용을 풍부하게**." [리포트 §1.2.3]

### 1-C. 직접 인용 vs 간접 인용
| 구분 | 정의 | 큰따옴표 | 출처 표기 | 사용 빈도 |
|---|---|---|---|---|
| 직접 인용 | 원문을 그대로(큰따옴표) 가져옴 | O | O | **최소화** |
| 간접 인용 | 이해한 내용을 내 언어로 다시 씀 | X | **반드시 O** | 대부분의 논문 |

- 직접 인용은 교수들이 "논문을 안 읽었거나 패러프레이징 노력을 안 한 것"으로 **부정적으로 평가** → 최소화. [리포트 §1.3 / 스크립트]
- 직접 인용 **허용 예외**: 원문 표현 자체가 너무 중요해 바꾸면 의미가 달라지는 경우 — **법률 조항, 공식 정의, 특정 개념의 최초 정의** 등. [리포트 §1.3.2 / 스크립트]
- 간접 인용 핵심: 큰따옴표는 안 쓰지만 **출처는 반드시 표기**. 단순 단어 치환이 아니라 "이해한 내용을 내 언어로 완전히 재서술". [리포트 §1.3.3]

### 1-D. "내 말로 쓰는 법" 4단계 (가이드의 핵심 실천법)
출처: [리포트 §2.1 / 스크립트]
1. 인용할 부분을 **이해될 때까지 최소 5~10번** 천천히 읽는다.
2. **논문(원문 창)을 닫고** 화면을 내린 뒤, 친구/대학교 1학년 학생에게 설명하듯 말로 해본다.
3. 다시 원문을 펴서 내가 설명한 내용과 **비교** — 빠진 부분/틀린 내용 수정.
4. 3↔4 과정을 여러 번 반복해 내용이 괜찮아지면 **마지막으로 출처를 표기**.
- 비유: "원문을 옆에 두고 쓰는 것 = 해설지 보며 수학 문제 풀기" → 실력이 안 늚. 뇌가 원문 문장 구조를 따라가게 됨. [스크립트]
- 연습 팁: 원본을 왼쪽에 쓰고 오른쪽 표에 다양한 방식으로 바꿔보기. "일주일만 해도 능숙해짐." [리포트 §1.3.4 / 스크립트]
- AI 패러프레이징: 직접 먼저 바꿔보고 이상할 때만 AI에게 확인. **AI가 만든 문장을 그대로 쓰면 안 됨**(논문 원표현 그대로 나오는 경우 많음). [리포트 §1.2.3 / 스크립트]

### 1-E. 간접 인용 3단계 비교 예시 (자료 원문 — 가상 예시)
출처: [리포트 §2.2 / 스크립트]
- **원문**: "스마트폰 과의존은 청소년의 학업 집중도를 저하시키고 수업 참여 의지를 감소시키는 것으로 나타났다."
- **❌ 표절에 가까운 버전(단어만 치환)**: "스마트폰에 지나치게 의존하는 것은 청소년의 수업 집중력을 떨어뜨리고 학습 참여 동기를 낮추는 것으로 확인되었다." → 문장 구조 동일, 유의어 치환 = 표절.
- **△ 불완전한 간접 인용(너무 뭉뚱그림)**: "이연구(2021)는 스마트폰 과의존이 학업에 부정적 영향을 준다고 했다." → 누가/누구에게/어떤 경로인지 불명확.
- **✅ 올바른 간접 인용(문장 전체 재작성)**: "이연구(2021)에 의하면 스마트폰 사용을 스스로 조절하지 못하는 청소년일수록 수업 중 집중을 유지하기 어려우며 학교 학습 활동에 능동적으로 참여하려는 의지도 낮아지는 경향이 있다." → 단어·구조 전면 변경 + 내용 풍부 + 핵심 의미 유지.
- 추가 연습 예시: "나는 채찍질 스터디에 참여하고 있다" → "나는 대학원생 연구자를 대상으로 2년간 계속 운영된 온라인 스터디에서 활동을 시작하였다." [리포트 §1.2.3]

### 1-F. 본문 인용 표기 (저자 수별, APA 기준) — 자료 원문
출처: [리포트 §3.1 / 스크립트]
- **저자 1명**: `홍길동(2020)에 의하면 ~` / `~ (홍길동, 2020).`
- **저자 2명**: `홍길동과 김연구(2020)는 ~` / `~ (홍길동, 김연구, 2020).`
- **저자 3명 이상**: `홍길동 외(2020)는 ~` / `~ (홍길동 외, 2020).`
- **복수 연구 동시 인용**: 세미콜론(;)으로 구분 — `(이연구, 2020; 김박사, 2021; 최연구, 2022)`.
- **학술지별 변형 주의**: 쉼표 대신 가운데점(·)·"and"·"&"를 쓰거나, 3인 이상도 첫 인용 시 전원 표기 후 두 번째부터 "외"를 쓰는 등 **투고 학술지 기준이 우선**. AI에게 "APA → 해당 형식 변환" 요청 가능하나 검토 필수. [리포트 §3.1.1 / 스크립트]

### 1-G. 재인용(2차 인용) — 지양 원칙
출처: [리포트 §3.1.3 / 스크립트]
- 재인용은 **원본을 끝내 찾을 수 없을 때의 최후 수단**. 원본을 직접 읽는 것이 학문적 정확성의 기본.
- 강의자는 "스터디 운영 중 재인용을 권장한 적이 한 번도 없다"고 명시. → 가이드에서도 **강하게 지양**으로 안내.
- ⚠️ **형식 검증 한계**: 자료에는 재인용의 **구체적 APA 표기 형식**(예: `원저자(원연도)는 ~라고 하였다(재인용 출처, 연도에서 재인용).`)이 제시되지 않음. 일반 통용 형식을 넣되 **"형식은 소속 지침 확인"** 디스클레이머로 표시(아래 §5 검증 메모 참조).

## 2. 인용 가이드 구조 (섹션 구성)

사용자 명시 흐름: **본문 인용 → 참고문헌 → 형식(APA/DOI) → 윤리**. 단, 자료의 무게중심(내 말로 쓰기·표절 회피)을 살려 다음 5섹션으로 구성한다. 참고문헌 목록 상세 형식은 기존 `apa-style` 페이지가 이미 보유하므로 **본 가이드는 요약 + 상호 링크**로 처리(중복 최소화).

```
/archive/citation-guide  (정적 단일 페이지, ArchiveStickyToc 5섹션)

1. 왜 인용하는가 · 표절이란            (id: why-cite)
   - 인용의 본질(어깨 위에 서기·근거 만들기)         [1-A]
   - 표절 3유형(미표기/복붙/유의어 치환)             [1-B]
   - "출처 달아도 구조 같으면 표절" 경고 박스

2. 본문 인용 — 직접 vs 간접            (id: in-text)
   - 직접/간접 비교표 + 직접 인용 허용 예외          [1-C]
   - 저자 수별 표기(1·2·3인↑·복수연구 세미콜론)     [1-F]
   - 페이지 표기(직접 인용 시 p./pp.)  ← apa-style 연계
   - 기관 저자/저자 없음/연도 없음 ← APA 통용(검증표시)

3. 내 말로 쓰는 법 — 4단계 + 비교 예시  (id: paraphrase)  ★핵심
   - 4단계 실천법(읽기→닫기→설명→비교→출처)        [1-D]
   - 간접 인용 3버전 비교(❌/△/✅) 카드             [1-E]
   - 연습 표 패턴(원본→변형) 안내                    [1-D]
   - AI 패러프레이징 주의                            [1-D]

4. 참고문헌 · 형식(APA/DOI) 요약        (id: references)
   - 4요소(저자·날짜·제목·출처) 요약               (apa-style 재사용)
   - 자료 유형별 형식은 → /archive/apa-style 링크
   - DOI 표기(https://doi.org/…)                    (apa-style 연계)
   - 표·그림 인용(출처 표기 규칙)  ← APA 통용(검증표시)

5. 인용 윤리 · 재인용                   (id: ethics)
   - 표절 = 윤리 위반(장기 리스크)                   [1-A]
   - 재인용 지양 원칙 + 형식 디스클레이머            [1-G]
   - 학술 책임 고지(지도교수·학술지 지침 우선)
```

### 섹션별 UI 컴포넌트 매핑(기존 재사용)
- 헤더: `PageHeader`(icon=`Quote` 또는 `BookMarked`)
- 우측 목차: `ArchiveStickyToc`(5섹션) — `apa-style`와 동일 2열 그리드 레이아웃
- 도입 안내: `InlineNotification kind="info"` (가상 예시·지침 우선 고지)
- 형식 예시: `apa-style`의 `RefBlock`(label/format/example) 패턴 차용
- 직접/간접 비교: `apa-style`의 `SectionTitle` + 비교표(table) 또는 좌우 카드
- 3버전 비교 예시: `writing-tips`의 ❌(rose)/✅(emerald) 색상 대비 카드 패턴 차용
- 윤리 고지: `writing-tips`·`paper-guide` 하단의 amber `AlertTriangle` 박스 패턴

## 3. 아카이브 반영 방식

### 3-1. 결론: **신규 정적 페이지** (신규 데이터 타입 불필요)
- `paper-guide`·`apa-style`가 이미 **정적 데이터 배열 + 단일 페이지** 패턴으로 운영 중. 인용 가이드도 콘텐츠가 큐레이션된 고정 지식이므로 동일 패턴이 가장 적합.
- ❌ **신규 Firestore 컬렉션/archive-seed 항목 불필요**: writing-tips처럼 staff CRUD가 필요한 가변 항목이 아님. 콘텐츠는 강의 자료 기반 고정. (추후 staff 편집 수요가 생기면 writing-tips와 같은 `citation_guides` 컬렉션으로 승격 가능 — 현 단계 비권장.)
- 데이터는 페이지 파일 내 `const SECTIONS`/`const EXAMPLES` 배열로 선언(= paper-guide의 `CHAPTERS`/`CAUTIONS` 방식).

### 3-2. 기존 `apa-style` 페이지와의 역할 분담 (중복 제거)
| 페이지 | 역할(조정 후) |
|---|---|
| `/archive/citation-guide` (신규) | **인용하는 "행위"**: 직접/간접, 표절 회피, 내 말로 쓰기, 본문 표기, 재인용, 윤리 |
| `/archive/apa-style` (기존) | **참고문헌 "형식"**: 자료 유형별 APA 7 reference list 형식, DOI, 6→7 변경점 |
- 두 페이지는 서로 **상호 링크**: citation-guide §2/§4에서 "자료 유형별 상세 형식은 APA 가이드 참조" 링크, apa-style 상단에 "본문 인용을 내 말로 쓰는 법은 인용 가이드 참조" 링크.
- 기존 apa-style의 §2(본문 내 인용)는 citation-guide §2와 일부 겹치나 apa-style은 형식 요약, citation-guide는 방법론 중심 — 의도적 분담으로 유지. (선택: 추후 apa-style §2를 축약하고 citation-guide로 위임.)

### 3-3. 진입점(링크) 연결
- `/archive` 랜딩(`src/app/archive/page.tsx`)의 "📖 학습·연구 가이드" 섹션에 **APA 가이드 카드 바로 앞 또는 뒤**에 "논문 인용 가이드" 카드 1개 추가. (apa-style 카드와 동일한 `<Link><article border-l-4>` 패턴 복제, 아이콘=`Quote`, 색상=teal/cyan 계열 제안 — rose/amber 이미 사용 중.)

## 5. 검증 메모 (자료 vs 가이드 — 미검증 형식 표시)

| 가이드 항목 | 자료 근거 여부 | 처리 |
|---|---|---|
| 인용의 본질·표절 3유형 | ✅ 자료 명시 | 그대로 반영 |
| 직접/간접 구분·직접 허용 예외 | ✅ 자료 명시 | 그대로 반영 |
| "내 말로 쓰기" 4단계 | ✅ 자료 명시 | 그대로 반영 |
| 간접 인용 3버전 비교 예시 | ✅ 자료 원문 | 가상 예시로 명기 |
| 저자 1·2·3인↑·복수연구(;) 표기 | ✅ 자료 명시 | 그대로 반영 |

| 자료 유형별 참고문헌 형식 | (apa-style 보유) | 신규 가이드는 요약 + 링크 |
