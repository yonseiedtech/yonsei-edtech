## 0. 범위 및 전제

- 본 설계는 **선행연구 정리 / 서론 작성 / 연구방법론 / 통계분석** 4영역의 아카이브 콘텐츠 고도화에 한정한다.
- **인용 방법(citation/APA)** 은 범위에서 제외하되, 선행연구 정리와 직접 맞닿는 최소 지점(간접인용·재인용·묶어쓰기 인용 표기)만 연계 언급한다. APA 형식 자체는 기존 `/archive/apa-style` 가 담당.
- **기존 아카이브 자산 재사용 최우선.** 신규 컬렉션/타입 신설은 최후 수단으로, 가능한 한 기존 `paper-guide`(정적), `research-method`/`statistical-method`/`foundation-term`/`writing-tip`(동적 시드) 를 확장한다.

## 1. 자료 핵심 요약 (4영역별 인사이트 추출)

### A. 선행연구 정리방법 (출처: [리포트] §2~3.2, [스크립트] 중반부)

1. **선행연구를 읽는 목적은 "내 연구가 왜 필요한지 증명"** — 익숙해지기 위함도 있으나 궁극적으로 인용·논리 구성용. (리포트 3.2-1)
2. **선행연구 정리 3대 질문** (리포트 3.2-2):
   - ① 이 연구는 **무엇을 밝혔는가** → 이론적 배경의 재료
   - ② 이 연구의 **한계는 무엇인가** → 내 연구의 필요성·독창성 근거
   - ③ **내 연구와 어떻게 연결되는가** → 서론 논리 흐름
3. **선행연구 정리표(심플 버전) 컬럼** (리포트 3.2-3): 저자·연도 / 연구주제 / 연구대상 / 연구방법 / 주요결과 / 한계점 / 내 연구와의 연결점.
   - 강의자 블로그의 풀 엑셀표는 "너무 길고 복잡" → 실제로는 위 심플 버전 권장. (스크립트)
4. **두세 편의 한계 비교만으로 연구모형이 나온다** (리포트 3.2-4, 핵심 사례):
   - 연구A: 스마트폰 과의존↔학업태도 (자기조절력 변수 **없음**)
   - 연구B: 자기조절력↔학업태도 (스마트폰 변수 **없음**)
   - → 두 한계의 교집합에서 "스마트폰+자기조절력 동시 독립변수" 연구모형 도출.
5. **읽고 요약만 하고 "연결"을 안 해서** 30편 읽어도 못 쓴다 (스크립트). 정리표의 핵심은 결과 요약이 아니라 한계→연결.
6. **간접인용 연습법(선행연구 활용과 직접 연계)**: 원문을 표 왼쪽에, 바꿔쓴 버전을 오른쪽에 두고 다양하게 변형. 일주일 연습으로 숙달. (리포트 1.3-4, 2.1)
7. **"내 말로 쓰기" 절차** (리포트 2.1-2, 표절 회피): ① 5~10회 정독 → ② 원문 닫기 → ③ 1학년에게 설명하듯 말/글로 재진술 → ④ 원문 다시 펴서 누락·오류 대조 → ⑤ ③④ 수회 반복 → ⑥ 출처 표기. 핵심: **원문을 보면서 쓰면 뇌가 구조를 따라가 표절 위험↑** (해설지 비유).

### B. 서론 작성법 (출처: [리포트] §3.3, [스크립트] 후반부)

1. **나열형 서론의 문제**: "A는~, B는~, C는~" 단순 나열은 "그래서 내 연구에 왜 필요한가"가 안 보임. (리포트 3.3-1)
2. **선행연구를 엮는 서론 4단계 흐름** (리포트 3.3-2, 핵심 프레임):
   - ① **현상 제시** — 사회적 문제 현상을 통계청 데이터·뉴스로 제시
   - ② **선행연구 소개** — 현상 관련 기존 연구 결과
   - ③ **연구 공백 지적** — 기존 연구 공통점·한계 분석으로 gap 명시
   - ④ **연구 목적 선언** — 이 공백을 메우겠다는 선언
   - → "살만 붙이면 한 페이지" / 외워두면 막힘이 준다.
3. **선행연구 묶어쓰기 3패턴** (리포트 3.3-4):
   - **유사결과 묶기**: 공통 결론을 한 문장으로, 세미콜론으로 다중 인용 — "…여러 연구에서 일관되게 확인되었다 (A, 2020; B, 2021; C, 2022)."
   - **상반결과 대비**: "반면", "이러한 상반된 결과는 …을 시사한다" 로 대조 후 원인 분석.
   - **공백 전환 표현**: "그러나/하지만/그럼에도 불구하고" 로 전환. ⚠️ **단, "그러나"를 싫어하는 교수가 많다**(강의자·강의자 지도교수 사례) → **검증 안 됨/개인 견해**. 지도교수 선호 파악 권장.
4. **개념정의·측정방법은 표로 제시** 하면 독자 이해를 돕는다. (리포트 3.3-4)

### C. 연구방법론 (출처: [리포트] §4.1, [스크립트])

1. **방법론을 먼저 정하면 연구가 막힌다 — 순서는 연구질문 먼저** (리포트 4.1-1·2, 핵심):
   - 연구질문이 명확해지면 방법론은 자연스럽게 따라온다.
2. **연구질문 유형 → 방법론 매핑** (리포트 4.1-3):
   - **영향력을 본다(숫자로 답)** → 양적(설문+통계). 예: "스마트폰 과의존이 학업태도에 미치는 영향은?"
   - **경험을 이해한다(이야기로 답)** → 질적(인터뷰·내러티브). 예: "…청소년은 학교생활을 어떻게 경험하는가?"
   - **둘 다** → 혼합. 시간·에너지 多 → 주로 박사논문. (리포트 4.1-3, "혼합=박사" 는 일반 경향이며 절대규칙 아님 — 다소 단순화)
3. **방법론은 한 번에 마스터 불가** — 방학마다 방법론책 1권 반복 정독 권장. (리포트 4.2-3, 학습전략 조언)

### D. 통계분석 (출처: [리포트] §4.2, [스크립트])

1. **회귀분석(Regression)** (리포트 4.2-1):
   - 용도: A→B 단순 인과. 독립변수 여러 개여도 **종속변수 1개**면 SPSS로 가능.
   - 예: "직무스트레스가 이직의도에 영향을 미치는가?"
   - 장점: 모형 심플·변수 적음 → **양적연구 입문에 최적**. 석사논문에서 다수 사용.
2. **구조방정식모형(SEM)** (리포트 4.2-2):
   - 용도: 다수 변수 간 **복잡한 경로를 동시 검증**. 회귀는 개별 관계만, SEM은 전체 경로 동시.
   - 예: 조직문화·직무만족·조직몰입·이직의도 전체 경로.
   - 도구: AMOS. **표본 최소 200명 이상**(미달 시 심사 지적). 심리상담·교육학 등 방법론 수준 높은 연구실은 석사논문에서도 SEM 사용.
3. **매개/조절/조절된 매개** 는 "다음 시간" 예고만 — 본 회차 미다룸. (확장 여지)
4. **인용 표기와 연계(통계 외 부분)**: 재인용은 최후수단(원본 직접 확인이 기본), 직접인용 최소화 — 본 설계 §A6·범위연계 지점.

> **검증 표시 요약**: "그러나 기피"(B-3), "혼합=박사 한정"(C-2), "SEM 200명"(D-2)은 강의자 경험·일반 관행 기반 조언으로, **학술적 절대 규칙이 아님**. 콘텐츠에 담을 때 "일반적 경향/지도교수·심사기준에 따라 다름" 단서를 병기한다. SEM 표본 200은 widely-cited 경험칙이나 모형 복잡도·추정법에 따라 달라짐을 명시.
