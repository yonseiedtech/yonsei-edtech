## 3. 구현 제안 (우선순위순)

### A. 학습이론 가계도(Theory Family) 레이어 — ROI 최고 ★추천
아카이브 개념에 `theoryFamily` 메타(행동주의계·인지주의계·구성주의계 + 세부 11계열)를 부여하고, **/archive에 "학습이론 지도" 뷰** 신설:
- 표지의 학자 네트워크처럼 **이론가—이론—파생 개념 그래프** (기존 자산 재사용: /research 계보도 렌더러·keyScholars·SEED_CONCEPT_VARIABLE_LINKS)
- 조건화 4형제(고전·결합·인접·작동), 구성주의 2계보(Piaget/Vygotsky) 같은 **혼동 페어를 시각적으로 구분** — 진단 compare 문항·기초용어 confusedWith와 연결
- 효과: 개념 68종이 낱개 카드에서 **"계보 속 위치"를 가진 지식 체계**로 승격. 종합시험·심사 대비 실효.

### B. 개념 시드 4차 보강 (갭 7종)
현재 아카이브에 없는 이 책의 핵심 계열: **고전적 조건화 · 시행착오 학습/효과의 법칙(Thorndike) · 인접 조건화(Guthrie) · 작동적 조건화(Skinner — 현 '행동주의'에서 세분) · 형태주의/통찰학습 · 잠재학습·인지도(Tolman) · 관찰학습/사회인지이론(Bandura)**.
- 정의는 패러프레이즈 + 원 학자 canonical 인용 + 이 책 페이지 인용 — **연세 저자 인용은 학회 정체성 강화 효과**
- 특히 관찰학습(Bandura)은 자기효능감(기존 개념)의 모이론인데 부재 — 최우선

### C. 이론 카드 "실천 방안" 섹션
각 이론 개념 상세에 "교수학습 실천 포인트" 필드(선택) 추가 — 책의 '이론→방안' 구조를 차용하되 내용은 자체 재서술 (예: 작동 조건화 → 프로그램 수업·행동목표·즉시 피드백 설계). 연구 인터뷰 theory-name 추천·진단 오답 학습 경로에서 함께 노출.

### D. 진단평가 "이론 계보" 문항
계보 구분형 compare 문항 신설: 고전 vs 작동 조건화(사전 자극 vs 결과 조형), 강화 vs 보상(책의 정밀 구분), 개인적 vs 사회문화적 구성주의, 통찰(형태주의) vs 시행착오(Thorndike), 잠재학습(Tolman)의 위치 등 — 시드 재서술 정의 기반 출제.
