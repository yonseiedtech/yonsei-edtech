## 핵심 갭 (우선순위)

### G1. 학습이론 기반 부재 (최우선)
개념 시드에 이론이 CTML 하나뿐. AECT '연구와 이론' 영역의 근간인
**행동주의(p.53)·인지주의 학습이론(p.79)·구성주의(p.107)·스키마이론(p.361)·정보처리이론(p.207)** 등이 없음.
→ 진단평가 개념 문항·이론카드(연구 인터뷰 theory-name 추천)의 커버리지 구멍.

### G2. 교수전략 계열 부재
스캐폴딩(p.359)·상황인지(p.380)·인지적 도제(p.71)·선행조직자(p.30)·문제 기반 학습(p.330)·프로젝트 기반 학습(p.342)·발견학습(p.144)·자기주도학습(p.364)·완전학습(p.273)·앵커드 교수법(p.38) 등.

### G3. 이러닝·정책 계열 부재
블렌디드 러닝(p.56)·원격교육 및 학습(p.146)·모바일 학습(p.290)·보편적 학습설계(p.404)·공개교육자료(p.313)·학습객체(p.256)·접근성(p.27).

### G4. 학습과정·분석 계열 부재
전이(p.401)·실천공동체(p.89)·요구사정(p.305)·경험의 원추(p.103)·귀인이론(p.30)·이중부호화이론(p.151)·분산인지(p.148).

### G5. 연구방법 갭
설계 기반 연구(DBR, p.125)·설계 개발 연구(p.123) — research-methods-seed 14종에 부재.

### G6. 용어 표기 불일치 (표준화 대상)
- 아카이브 "Assessment=평가" 혼용 ↔ AECT 공식: **Assessment=사정 / Evaluation=평가** (기초용어 measurement 카테고리에서 명시 구분 필요)
- "학습공동체" ↔ AECT "실천공동체(Community of Practice)" — altNames/aectTerm 병기로 해소
- 관행 표기(예: 플립러닝·SAMR·학습분석 등 2013년 이후 용어)는 AECT 미수록 — aectTerm 없음이 정상

## 반영 계획 (A~E + 추가 기능)

1. **A. 표기 표준화**: `aectTerm` 필드(ArchiveConcept·FoundationTerm) 추가 — purifiedName 병기 패턴 재사용. 시드에 공식 역어 채움.
2. **C. 참고문헌 보강**: AECT 표제어와 매칭되는 기존 개념 references에 책+표제어 페이지 추가.
3. **B. 신규 시드**: 위 G1~G4에서 ~24개 개념 신규 추가(본문 판독→패러프레이즈), G5 연구방법 2종 추가.
4. **D. 설명 개선**: 기존 개념 설명 번역투 순화 + 진단 concepts 영역과 개념 커버리지 정렬(conceptSeedKey).
5. **E. 인용 가이드**: 번역서 APA 인용(원서/역서 이중 표기) 예시 추가.
6. **추가 기능**: 아카이브 "AECT 용어 표준 사전" — 186개 매핑 검색·영역 필터·아카이브 개념 연결 (src/lib/aect-terminology.ts 데이터 모듈).
