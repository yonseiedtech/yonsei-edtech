## 3. 참가자 투표 반영 방식 (X1 후단)

- 심사단 루브릭 점수(20점 만점)와 참가자 투표 수는 **별도 표기** — 콘솔 심사 탭에서 나란히 확인.
- 최종 수상 결정 시 두 지표를 어떻게 합산할지(예: 심사 80% + 투표 20%, 또는 "인기상" 별도 신설)는 **수상 지정 전 운영진 판단** — 시스템은 강제 합산하지 않음.
